# LatamCreativa — Tipografías

## Fuente Principal
- **Nombre:** Inter
- **Fuente:** Google Fonts (next/font/google)
- **Variable CSS:** --font-inter
- **Subsets:** Latin
- **Display:** Swap

## Escala Tipográfica (Tailwind Default)

| Clase         | Tamaño   | Line-height | Uso típico                    |
|---------------|----------|-------------|-------------------------------|
| text-xs       | 0.75rem  | 1rem        | Badges, labels pequeños       |
| text-sm       | 0.875rem | 1.25rem     | Texto secundario, captions    |
| text-base     | 1rem     | 1.5rem      | Texto body principal          |
| text-lg       | 1.125rem | 1.75rem     | Subtítulos, descripciones     |
| text-xl       | 1.25rem  | 1.75rem     | Títulos de cards              |
| text-2xl      | 1.5rem   | 2rem        | Títulos de sección            |
| text-3xl      | 1.875rem | 2.25rem     | Títulos principales           |
| text-4xl      | 2.25rem  | 2.5rem      | Headings grandes              |
| text-5xl      | 3rem     | 1           | Hero subtítulo                |
| text-6xl      | 3.75rem  | 1           | Hero título                   |
| text-7xl      | 4.5rem   | 1           | Display grande                |
| text-8xl      | 6rem     | 1           | Display extra grande          |
| text-9xl      | 8rem     | 1           | Display máximo                |

## Hero (tipografía fluida)
- Fórmula: clamp(3rem, 8vw, 5.5rem)
- Font-weight: 200 (Extra Light)
- Letter-spacing: tight
- Line-height: 0.95

## Pesos de Fuente Usados

| Clase            | Weight | Uso                                |
|------------------|--------|------------------------------------|
| font-extralight  | 200    | Hero títulos                       |
| font-light       | 300    | Descripciones hero, párrafos       |
| font-normal      | 400    | Texto body                         |
| font-medium      | 500    | Labels, botones, tabs              |
| font-semibold    | 600    | Títulos de cards, headings         |
| font-bold        | 700    | Títulos importantes, CTAs          |

## Link Google Fonts (para uso externo)
```
https://fonts.googleapis.com/css2?family=Inter:wght@200;300;400;500;600;700&display=swap
```

## CSS Import (alternativo)
```css
@import url('https://fonts.googleapis.com/css2?family=Inter:wght@200;300;400;500;600;700&display=swap');
```
